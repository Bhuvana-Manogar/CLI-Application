# CLI-Application

This project involves creating a CLI application that integrates with a flight data API such as Aviationstack. Users can input commands to search flights by number, route, or date, and view stored flight information. The system will periodically fetch updates, store them in a database, and allow filtering through CLI commands. This project simulates real-world airline backend systems without a graphical interface.
